# Jidi Urhi Game (C++)

A multithreaded simulation of the traditional "Jidi Urhi (Bird Fly)" game using C++, POSIX Threads (Pthreads), Mutexes, and Semaphores.

## Features
- 5 Player Threads
- Announcer Thread
- Referee Thread
- Random Bird/Non-Bird Announcements
- Player Elimination
- Winner Detection
- Survival Statistics

## Technologies
- C++
- POSIX Threads (pthread)
- Mutex
- Semaphores

## How to Compile

```bash
g++ main.cpp -o game -pthread
```
